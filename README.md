# Visual Effect Parameter Optimizer

Built for HarmonyOS. Based on genetic algorithms.

## Setup Requirement
- install [uv](https://docs.astral.sh/uv/getting-started/installation/#standalone-installer) for Python package management.
- install openharmony toolchain from [CI](https://ci.openharmony.cn/workbench/cicd/dailybuild/dailylist). Add `hdc` to shell PATH.
